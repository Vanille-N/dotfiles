# Dotfiles

Dotfiles for my Arch [i3] setup.
This whole repo is essentially the non-sensitive part of my `.config/`

## Includes
```
──┬── bottom/bottom.toml
  ├── dunst/dunstrc
  ├── env/
  │    ├── alias
  │    ├── bashrc
  │    ├── funcs
  │    ├── keyboard
  │    ├── vars
  │    ├── vimrc
  │    ├── xcompose
  │    └── xinitrc
  ├── i3/config
  ├── i3status/config
  ├── kitty/
  │    ├── kitty.conf
  │    └── theme.conf
  ├── mozilla/
  │    ├── chrome/*
  │    ├── icons/*
  │    ├── newtab/*
  │    ├── scripts/*
  │    ├── index.html
  │    └── style.css
  ├── neofetch/config.conf
  ├── oomox/colors/greenified-gruv
  ├── qutebrowser/config.py
  ├── rofi/powermenu/
  │    ├── styles/gruv.rasi
  │    ├── full_square.rasi
  │    └── sys-menu.sh
  ├── scripts/
  │    ├── alt-paste
  │    ├── apps-menu

```
